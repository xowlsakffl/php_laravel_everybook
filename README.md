# 에브리북 📚
개인 프로젝트 SNS 에브리북.

## 개요 📌
![11](https://user-images.githubusercontent.com/50791439/194874905-38417580-0aff-4a89-85f5-53c531c8d513.PNG)
포스트와 피드 작성할 수 있는 SNS<br>
SNS에 필요한 여러 기능 탑재

## 주요 기능
<ul>
  <li>커스텀 로그인, 회원가입</li>
  <li>팔로우, 팔로워</li>
  <li>댓글, 대댓글</li>
  <li>좋아요 기능</li>
  <li>파일 업로드</li>
  <li>채팅</li>
</ul>

## 사용 스택 ⚒️
<ul>
  <li>Laravel 9</li>
  <li>mysql</li>
  <li>tailwindcss</li>
  <li>dropzone(드래그앤 드랍 이미지 업로드 패키지)</li>
  <li>커스텀 인증</li>
</ul>
